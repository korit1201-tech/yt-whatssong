# Chrome Extension 安裝與使用說明

這是一個專為你打包的 Chrome 擴充功能，請依照以下步驟安裝給朋友使用。

## 1. 安裝步驟

由於此擴充功能尚未上架至 Chrome 線上應用程式商店，需要使用「開發者模式」進行安裝。

1.  **解壓縮檔案**：
    將收到的 `.zip` 壓縮檔解壓縮到電腦上的一個資料夾中。

2.  **開啟擴充功能管理頁面**：
    在 Google Chrome 瀏覽器的網址列輸入 `chrome://extensions` 並按下 Enter。
    或者從選單進入：右上角「⋮」 > 「擴充功能」 > 「管理擴充功能」。

3.  **開啟開發者模式**：
    在頁面右上角，找到「開發者模式」開關並**開啟**。

4.  **載入擴充功能**：
    點擊左上角出現的「載入未封裝項目」按鈕。
    選擇**步驟 1 解壓縮後的資料夾**（請確保選擇包含 `manifest.json` 的那一層資料夾）。

5.  **完成**：
    現在你應該能在擴充功能列表中看到這個外掛，並且可以使用了！

## 2. 使用說明

- **固定圖示**：建議在瀏覽器右上角「拼圖」圖示中，將此擴充功能的「圖钉」按下去，讓它固定在工具列上方便操作。
- **功能操作**：點擊圖示即可打開控制面板，依照介面上的選項進行設定。
- **權限**：如果瀏覽器詢問權限，請點擊「允許」以確保功能正常運作。

如果有任何問題，請隨時聯繫！
